# コメダセット暗記 (Komeda Set Anki)

コメダ珈琲店での注文に対し、提供すべきパーツ（カップ、ソーサー、豆菓子、スプーンなど）の正しい組み合わせを覚えるためのトレーニングアプリです。

## 🌟 特徴

- **PWA 対応**: スマホのホーム画面に追加して、オフラインでもネイティブアプリのように使えます。
- **個別トレイ判定**: 複数の注文（例：ホットコーヒーとアイスコーヒー）に対し、それぞれのトレイに正しいアイテムが載っているかを個別に判定します。
- **管理機能**: パーツ（アイテム）の登録や、注文と正解セットの紐付けをアプリ内から自由に行えます。
- **データ永続化とバックアップ**: 登録したデータはブラウザに保存されるほか、JSON形式でエクスポート・インポートが可能です。
- **暗記モード**: 不正解時は正解を隠した状態で「やり直し」ができ、効率的に覚えられます。

## 🚀 使い方

### 1. クイズに挑戦する
1. 画面上部に表示される「注文伝票」を確認します。
2. 準備したい注文カードをタップして選択します（赤い枠がつきます）。
3. 下部のメニューから、その注文に必要なアイテムをタップしてトレイに載せます。
4. すべての注文の準備ができたら「提供 (Serve)」ボタンを押します。
5. 全て正解なら次の問題へ、不正解なら「やり直す」または「正解を確認する」を選択します。

### 2. データを管理・編集する
1. ヘッダーの ⚙️ アイコンをタップして管理ダッシュボードを開きます。
2. **パーツ管理**: コップや皿、消耗品などの画像をアップロードして登録できます。
3. **注文セット管理**: 「コメダブレンド」などの商品名に対し、どのパーツが必要かを設定します。
4. **データ管理**: 登録した全データをファイルとしてバックアップしたり、別の端末に移行したりできます。

### 3. スマホでアプリとして使う
1. ブラウザ（Safari/Chrome）でアプリのURLを開きます。
2. iPhoneの場合：Safariの「共有」ボタン → 「ホーム画面に追加」を選択。
3. Androidの場合：Chromeのメニュー → 「ホーム画面に追加」を選択。

## 🛠 開発者向けセットアップ

このプロジェクトは React と Vite で構成されており、パッケージ管理には [pixi](https://pixi.sh/) を使用しています。

### 起動方法
```bash
# 依存関係のインストールと起動
pixi run dev
```

### ビルド
```bash
# 公開用ファイルの生成
pixi run build
```
